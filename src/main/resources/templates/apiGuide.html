 - API FUNCIONANDO CORRECTAMENTE -

Endpoints:

Productos

"/productos/crear": Permite crear de a un producto a la vez
"/productos/traer": Devuelve el total de productos en la BD
"/productos/batch": Permite crear productos en base a un array
"productos/borrar/{id}": Permite eliminar un producto indicando su ID
"productos/editar/{id}": Permite editar un productos indicando su ID

Publicaciones Mercadolibre

"/ml/crear": Permite crear de a una publicación a la vez
"/ml/traer": Devuelve el total de publicaciones en la BD
"/ml/batch": Permite crear publicaciones en base a un array
"/ml/borrar/{id}": Permite eliminar una publicación indicando su ID
"/ml/editar/{id}": Permite editar una publicación indicando su ID

Publicaciones pagina WEB

"/web/crear": Permite crear de a una publicación a la vez
"/web/traer": Devuelve el total de publicaciones en la BD
"/web/batch": Permite crear publicaciones en base a un array
"/web/borrar/{id}": Permite eliminar una publicación indicando su ID
"/web/editar/{id}": Permite editar una publicación indicando su ID
