 - API FUNCIONANDO CORRECTAMENTE -

Endpoints:

Productos

POST "/productos/crear": Permite crear de a un producto a la vez
GET "/productos/traer": Devuelve el total de productos en la BD
POST "/productos/batch": Permite crear productos en base a un array
DELETE "productos/borrar/{id}": Permite eliminar un producto indicando su ID
PUT "productos/editar/{id}": Permite editar un productos indicando su ID

Publicaciones Mercadolibre

POST "/ml/crear": Permite crear de a una publicación a la vez
GET "/ml/traer": Devuelve el total de publicaciones en la BD
POST "/ml/batch": Permite crear publicaciones en base a un array
DELETE "/ml/borrar/{id}": Permite eliminar una publicación indicando su ID
PUT "/ml/editar/{id}": Permite editar una publicación indicando su ID

Publicaciones pagina WEB

POST "/web/crear": Permite crear de a una publicación a la vez
GET "/web/traer": Devuelve el total de publicaciones en la BD
POST "/web/batch": Permite crear publicaciones en base a un array
DELETE "/web/borrar/{id}": Permite eliminar una publicación indicando su ID
PUT "/web/editar/{id}": Permite editar una publicación indicando su ID
